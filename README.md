# 🎓 Student Grade Management System

## 📌 Description
A console-based Student Grade Management System built using Core Java.

## ✨ Features
- Enter Student Name
- Enter Marks for 3 Subjects
- Calculate Average
- Display Grade (A/B/C/Fail)
- Display Student Report

## 🛠 Technologies Used
- Java
- Scanner Class
- Arrays
- Conditional Statements
- Classes
